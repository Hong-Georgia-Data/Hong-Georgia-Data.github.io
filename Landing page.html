Explanation




This is my first individual attempt of a website created for a class assignment. Weather infomation from around 500 to 600 random cities are extracted from an accessible API. This is to demonstrate capability in performing an API calls with Python. Next, by using Python Pandas library compiling those datas into csv files and pandas dataframes for easier processing. Third step, using matplotlib library, creating several graphs for more intuitive visual analyis. The graphics on this site is the results of that step.

The API is only accessible to a certain point. If we perform a call that has a resulted of over 60 cities, it would locked the user until a lengthy period is passed. Accessibility is limited to only the current datetime when it was called, past data is inside a paywall. To keep results consistent, only information from the date 8/25/2020, is collected.

Data from Open Weather API on 8/25/20.
